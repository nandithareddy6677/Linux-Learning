# 📖 Introduction

Linux powers most cloud infrastructure worldwide.

The majority of cloud servers, web servers, containers, and enterprise applications run on Linux because of its stability, security, and flexibility.

---

# ☁️ Linux in the Cloud

Linux is widely used by cloud providers.

Examples:

- AWS EC2
- Azure Virtual Machines
- Google Compute Engine

---

# Common Linux Cloud Distributions

- Ubuntu Server
- Amazon Linux
- Red Hat Enterprise Linux (RHEL)
- Debian
- CentOS
- Rocky Linux

---

# Typical Cloud Administration Tasks

- Launch Virtual Machines
- Configure SSH Access
- Manage Users
- Configure Firewalls
- Monitor System Health
- Install Software
- Update Packages

---

# SSH in Cloud

Administrators securely access cloud servers using SSH.

Example:

```bash
ssh ubuntu@<public-ip>
```

---

# Storage

Cloud Linux servers use:

- EBS Volumes (AWS)
- Managed Disks (Azure)
- Persistent Disks (Google Cloud)

---

# Networking

Linux cloud servers interact with:

- VPC
- Security Groups
- Route Tables
- Firewalls
- Load Balancers

---

# Monitoring

Administrators monitor:

- CPU
- Memory
- Disk
- Logs
- Network

Tools:

- top
- htop
- journalctl
- CloudWatch
- Azure Monitor

# Linux Skills Required for Cloud Engineers

- Command Line

---

# 🌍 Real-Life Example

A Cloud Engineer launches an Ubuntu EC2 instance, connects using SSH, installs Nginx, configures the firewall, and deploys a web application.

---

# 💡 Best Practices

- Keep Linux systems updated.
- Use SSH keys.
- Restrict Root access.
- Monitor logs.
- Regularly back up cloud instances.

---

# 📌 Key Takeaways

- Linux powers most cloud infrastructure.
- Cloud Engineers use Linux daily.
- SSH is the primary administration method.
- Linux administration skills are essential for AWS, Azure, and Google Cloud.

# ❓ Interview Questions

### 1. Why is Linux widely used in cloud computing?

Because it is secure, stable, open-source, and highly customizable.

### 2. Name three Linux distributions commonly used in the cloud.

- Ubuntu
- Amazon Linux
- Red Hat Enterprise Linux

### 3. Which protocol is commonly used to manage Linux cloud servers?

SSH.

### 4. Name one AWS service that runs Linux virtual machines.

Amazon EC2.

### 5. Why are Linux skills important for Cloud Engineers?

Most cloud infrastructure and servers run Linux.
